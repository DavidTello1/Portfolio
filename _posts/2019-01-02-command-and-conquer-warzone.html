Role

This project was made for a university subject in a team of 7.

My role in this project was to be the programmer, my main mission was to make a good and well structured code for the game and ensure other team members followed the coding guidelines.

I was in charge of coding the main core of the game and first versions of new features as well as to help other teammates with code related issues.


Features

Core

Creating the base and preparing it to be used to start coding the game.

- UI System: Implement the first version of the UI system and code the actions and logic.

- Entity Manager: Create the entity manager module, a factory class to manage all the entities in the scene.

- Buildings: Create the first versions of the buildings.


Build System

Implement the system to place and build buildings.


Sprite Ordering

Implement sprite ordering to draw the entities with the correct depth


Healthbars

Added healthbars to every entity to shows its current health.
